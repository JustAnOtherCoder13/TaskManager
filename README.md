## 3. Work done :

### Clean architecture.
+ Single activity.
+ Multi module.
+ DI with Hilt.
+ Coroutines and Flow.

### Compose.
+ Entire Ui with jetpack compose.
+ Only one activity, no fragment and no business logic in application module

### Other.
+ SOLID/DRY approach.
